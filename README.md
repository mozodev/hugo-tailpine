# Hugo theme tailpine

Hugo theme using tailwindcss and alpinejs.

## development

```
# Run server.
$ go version
go version go1.17 linux/amd64 (> 1.12)
$ git clone https://github.com/team-durumi/hugo-theme-tw.git
$ cd hugo-theme-tw/exampleSite
$ npm i && hugo serve
# open http://localhost:1313
```
